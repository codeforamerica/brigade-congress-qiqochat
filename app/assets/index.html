How to use QiqoChat

QiqoChat is a new event platform that connects note documents with Zoom breakout rooms. The buttons along the left side are the various rooms to join, and each room has a notetaking Google Doc (or other links) in the space above this area.

Please check out the Attendee Toolkit for more information about how to use QiqoChat.
